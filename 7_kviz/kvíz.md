# Projekt: Kvíz hra

## Popis projektu
Tento projekt je jednoduchá kvízová hra vytvořená v programovacím jazyce Python. Hráč odpovídá na otázky a získává body za správné odpovědi.

## Funkce programu
- zobrazení otázek
- více možností odpovědí
- kontrola správnosti odpovědí
- počítání bodů
- závěrečné vyhodnocení skóre

## Jak program funguje
Otázky jsou uloženy v seznamu. Program postupně vypisuje otázky a možnosti odpovědí. Hráč zadává odpovědi pomocí písmen a, b nebo c. Za každou správnou odpověď získá bod.

## Použité technologie
- Python 3

## Spuštění programu
Program se spustí příkazem:

```bash
python kviz.py
